## Intro
2D + 탑 다운 도트 게임

시작 위치  :  병원 3층

각 챕터마다 ‘보스 좀비’ 를 처치하면 다음 챕터로 넘어갈 수 있음

병원 -> 도로 -> 연구실


#### 조작법
이동 : WASD, SHIFT : 달리기, F : 상호작용, SPACE : 대쉬, 마우스 좌클릭 : 공격
